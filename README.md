# MoneyPylot
MoneyPylot is a python app designed to monitor and manage your income and expenses, empowering you to maintain effective control over your financial health. With this tool, users can effortlessly categorize their expenditures, set budgets, and gain a comprehensive understanding of their financial landscape.


The goal of this project is to manage:
- Monthly income (recurring and variable)
- Custom categories and subcategories
- Monthly expenses
- Recurring expenses and installment payments
- Budgeting and spending analysis

This project is currently in early development.  
All logic is being built for the console version first.  
A GUI will be added later.

## Tech Stack
- Python 3.x
- SQLite3 (local database)
- No external dependencies for the core logic
